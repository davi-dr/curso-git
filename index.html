Olá, Mundo Git! 👋

Comandos

  1. git init - Inicia um novo repositório vazio no GIT local na pasta atual.
  2. git status - Verifica o status atual do repositório, incluindo arquivos modificados e arquivos na (Staging Area).
  3. git commit -m = Grava as alterações feitas na área de stage como uma nova versão do código.
  4. git add + NomeArquivo - Adiciona arquivos selecionados para a área de stage (pronto para ser commitado).
  5. git add . - Adicionar todos os arquivos.
  6. git log - Ver o histórico de commits.
